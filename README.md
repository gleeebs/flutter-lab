# EasyCalc 

## Краткое описание
EasyCalc — мобильное приложение-калькулятор с конвертером валют и историей операций.

## Основные функции
- Калькулятор: базовые операции
- Конвертер валют: перевод сумм по актуальному курсу (используется публичное API)
- История: сохранение всех выполненных операций
- Переключение между режимами через навигацию

## Структура экранов

**Экран 1: Калькулятор**
- Дисплей (ввод и результат)
- Кнопки: цифры, операции
- Нижняя навигация (Калькулятор | Конвертер)
- Кнопка "История" в шапке

**Экран 2: Конвертер**
- Поле ввода суммы
- Выбор валют 
- Кнопка "Конвертировать"
- Отображение результата
- Нижняя навигация
- Кнопка "История" в шапке

**Экран 3: История**
- Заголовок: "История"
- Список операций
- Кнопка "Очистить всё"
- Кнопка "Назад"

## Навигация
- Нижнее меню: переключение между калькулятором и конвертером
- Кнопка "История" открывает новый экран поверх текущего
- Из истории возврат по стрелке "Назад"
